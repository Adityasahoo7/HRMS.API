==============================================================
HRMS — PER MODULE LAYER BREAKDOWN
DAL | BLL (DTOs + Validators) | Service | Controller
==============================================================

NOTE:
  [NEW] = Newly added module not in original blueprint
  Every module shares these across all layers:
    DAL SHARED    → IGenericRepository, GenericRepository, IUnitOfWork, UnitOfWork, AppDbContext
    BLL SHARED    → MappingProfile.cs
    SHARED LIB    → ApiResponse, PagedResponse, AppConstants, RoleConstants,
                    NotFoundException, BadRequestException, UnauthorizedException, ForbiddenException
    MIDDLEWARE    → ExceptionMiddleware, JwtMiddleware, RequestLoggingMiddleware
